# Demo1

This is a Spring Boot application that demonstrates the use of various Spring modules and JWT for security.

## Technologies Used

- Spring Boot 3
- Spring Security 6
- Spring Data JPA 3
- Spring Data REST 3
- JWT
- Lombok
- Maven
- PostgreSQL
- Springdoc OpenAPI
- Swagger UI
- RabbitMQ

## Installation

To install the project, clone the repository and run the following command:

```bash
mvn clean install
```
